## 📌 Tip Calculator

Helps users calculate tip amount and total bill split based on percentage input.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Calculates tip amount based on bill value and percentage.
* Supports bill splitting among multiple people.
* Provides quick and clear output for better usability.
